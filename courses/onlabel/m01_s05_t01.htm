There are three basic interactive elements: popups, subscreens, and knowledge checks. We will cover subscreens on the following page and knowledge checks in a subsequent section.

A popup is a small area that appears on the screen. It may provide a definition or other limited information. Popups can contain graphics or icons, but keep in mind that popups are typically fairly small.

Customizing popups

Popups can be customized for specific needs beyond a standard popup without having to modify the JavaScript. You can customize a popup by defining a style name during entry and modifying tooltips.css to define the specific customization. The intent is to define the visual styles in the CSS so it can be reused consistently throughout a course or several courses for a customer while keeping entry simple. If there is a special need to customize one popup that needs to be unique it can be done in i-prism. Otherwise it is more efficient to define the style once in the CSS and reuse that style multiple times. Here are some examples of customized popups for sample entry and CSS entry.

The position of a popup can be customized. By default it is place near the mouse cursor. The position can be added as x and y coordinates or by percentages based on the top left of the window.
