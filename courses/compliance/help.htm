Titles

The titles are displayed at the top left of the screen in the following order:

  • Module title
  • Section title: Topic title
Global Navigation

 The Table of Contents lists all screens in the module.

 Help displays this help screen.

Transcript

 The Transcript icon my be used to display the transcript window, which contains the text of the audio. To close the transcript, select the icon again or select the in the upper right of the transcript window.

Media Controller

 The Media Controller offers control over the playback of media including audio and screen sequences.

 The Play or Pause icon may be used to pause and resume playback. Selecting Play when the playback counter is at 0 will cause the screen to play again.

Navigation and Prompts

 The Back and Next icons are used to navigate through the module. When active, an instructional prompt will appear above them to indicate the next step.

 Between the icons is the progress indicator.

Exit Module

 Select Exit at the bottom right of the screen to close the module.
